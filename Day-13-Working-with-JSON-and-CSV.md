# 📖 Introduction

Many applications exchange data using JSON and CSV.

Python provides built-in modules to process these formats efficiently.

---

# What is JSON?

JSON (JavaScript Object Notation) stores structured data.

Example

```json
{
  "name":"Alice",
  "age":22
}
```

---

# JSON Module

```python
import json
```

---

# Convert Python to JSON

```python
json.dumps()
```

---

# Convert JSON to Python

```python
json.loads()
```

---

# Reading JSON File

```python
json.load()
```

---

# Writing JSON File

```python
json.dump()
```

---

# CSV Module

```python
import csv
```

---

# Reading CSV

```python
csv.reader()
```

---

# Writing CSV

```python
csv.writer()
```
